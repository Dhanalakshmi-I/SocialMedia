# Social Media Application

## Description

This project is a simple **Social Media backend application** developed using Java and Spring Boot.
It demonstrates how a basic social media system works by allowing users to create accounts, post content, and interact with posts.

The application mainly focuses on **REST API development and database integration**.

## Features

* User registration and login
* Create and manage posts
* View posts from other users
* Like and comment on posts
* Store and retrieve data from database

## Technologies Used

* Java
* Spring Boot
* Maven
* REST API
* Database (MySQL / H2)

## How to Run

1. Clone the repository
2. Open the project in an IDE like Eclipse or IntelliJ
3. Install dependencies using Maven
4. Run the main class `SocialMediaApplication`
5. Test APIs using Postman

## Future Enhancements
